PC-ZEN

개설 년도 : 1994년


주요 전공 : C/JAVA/C#/C++


동아리 인원 : 2020년 신학기 (2020.03.02)기준 15명(2학년 7명 3학년 8명)


동아리 임원 : 기장 : 최영하 부기장 : 심재현




동아리 소개 : PC-ZEN은 올해로 27기 인원을 모집하는 역사가 살아 숨 쉬는 동아리입니다. 기초적인 프로그래밍부터 전문 프로그래밍 언어까지 공부 할 수 있으며,


동아리 내에서 다양한 활동을 통해 프로젝트를 진행하고, 자신의 진로를 찾을 수 있습니다. 또한 자기 주도 학습 능력을 키울 수 있는 기회가 제공되며,


동아리 내의 분위기는 때론 가족 같고, 때론 회사 같으며 때론 즐거운 동아리입니다. 특히 전교에 아침에 활동하는 유일한 동아리로,


시간을 효율적으로 사용하며 공부를 하고자 하는 학생분들께 추천하는 동아리 입니다.




동아리 활동 시간 : 07:30~08:40/17:10~21:00 (18:30~19:20 석식 시간)


동아리 활동 장소 : 본관 4층 프로그래밍 2실


ROGIC

개설 년도 : 2012년

주요 전공 : Robot C/Arduino/Raspberry Pi


동아리 인원 : 2020년 신학기 (2020.03.02.)기준 11명(2학년 9명 3학년 3명)


동아리 임원 : 기장 : 채현서 부기장 : 김승호




동아리 소개 : Rogic은 유일한 하드웨어 동아리로 기초적인 IOT 프로그래밍을 배울 수 있고 동아리원들과 팀을 맺어 프로젝트를 진행해 협동심을 기를 수 있게 도와줍니다.


동아리 내 분위기가 매우 좋아 선후배간 친밀감도 좋아 상호간의 거리낌도 없애주기도 합니다.


비교적 자유로운 분위기에 적응하기에도 쉬우며, 지역 축제에 많이 참여하여 자신의 포트폴리오를 화려하게 장식하고 싶은 분들에게 추천하는 동아리입니다.




동아리 활동 시간 : 17:20~21:00 (18:30~19:30 석식 시간)


동아리 활동 장소 : 본관 3층 멀티미디어실


TAG

개설 년도 : 2016년


주요 전공 : HTML/CSS/JavaScript


동아리 인원 : 2020년 신학기 (2020.03.02.)기준 (2학년 6명 3학년 8명)


동아리 임원 : 기장 : 이서연 부기장 : 공석




동아리 소개 : TAG는 웹프로그래밍과 디자인/그래픽 동아리로 기초적인 HTML 언어를 배울 수 있고, 많은 실습을 통하여 실력을 쌓을 수 있으며,


팀 단위 프로젝트를 통하여 실력을 뽐낼 수 있습니다.


자유로운 분위기와 차분한 분위기가 적재적소하게 공존하는 동아리로 적응하기에도 쉬우며 공부하는 분위기가 조성됩니다.


웹 분야와 그래픽/디자인 등을 실무 위주로 공부하고 싶은 학생분들에게 추천하는 동아리 입니다.




동아리 활동 시간 : 17:00~21:00 (18:30~19:20 석식 시간)


동아리 활동 장소 : 본관 4층 프로그래밍 1실
